# 🚀 TaskFlow - MERN Stack Task Manager

TaskFlow is a modern and responsive MERN Stack Task Management application that helps users organize, manage, and track their daily tasks efficiently. It provides a clean user interface with secure authentication and real-time task management.

---

## 🌐 Live Demo

🔗 https://taskflow-mern-stack-ecru.vercel.app

## 📂 GitHub Repository

🔗 https://github.com/nidasheikhh/taskflow-mern-stack

## ✨ Features

- 🔐 User Authentication (Login & Register)
- 📋 Create Tasks
- ✏️ Update Tasks
- 🗑 Delete Tasks
- ✅ Mark Tasks as Complete
- 📅 Organize Daily Tasks
- 📱 Responsive Design
- ⚡ Fast and User-Friendly Interface

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/nidasheikhh/taskflow-mern-stack.git
```

Move into the project

```bash
cd taskflow-mern-stack/mern-taskflow
```

---

## Install Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## Install Backend

Open another terminal

```bash
cd backend
npm install
npm start
```

---

## Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=5000

MONGODB_URI=Your MongoDB Connection String

JWT_SECRET=YourSecretKey
```
